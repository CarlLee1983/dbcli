```typescript
export function levenshteinDistance(a: string, b: string): number {
  const matrix: number[][] = []

  // Initialize matrix
  for (let i = 0; i <= b.length; i++) {
    matrix[i] = [i]
  }
  for (let j = 0; j <= a.length; j++) {
    matrix[0][j] = j
  }

  // Fill matrix with dynamic programming
  for (let i = 1; i <= b.length; i++) {
    for (let j = 1; j <= a.length; j++) {
      if (b.charAt(i - 1) === a.charAt(j - 1)) {
        matrix[i][j] = matrix[i - 1][j - 1]
      } else {
        matrix[i][j] = Math.min(
          matrix[i - 1][j - 1] + 1,  // Substitution
          matrix[i][j - 1] + 1,      // Insertion
          matrix[i - 1][j] + 1       // Deletion
        )
      }
    }
  }

  return matrix[b.length][a.length]
}
```

Function signature:
```typescript
export async function suggestTableName(
  errorMessage: string,
  adapter: DatabaseAdapter
): Promise<{suggestions: string[]; tables: string[]}>
```

Implementation logic:
1. Extract quoted table name from error message using regex
   - Match patterns: relation "users", table 'users', `users` (PostgreSQL and MySQL)
   - Regex: /relation\s+['""`](\w+)['""`]|table\s+['""`](\w+)['""`]/i
   - Extract the captured group (the table name that failed)

2. Call adapter.listTables() to get all available tables

3. Compute Levenshtein distances between failed table name and all available tables

4. Filter: keep only distances < 3 (likely typos)

5. Sort by distance ascending (closest first)

6. Return top 3 suggestions

7. Return object: { suggestions: string[], tables: string[] }
   - suggestions: closest matches (max 3)
   - tables: all available table names (for fallback listing)

Error handling:
- If no table name can be extracted, return empty suggestions but list all tables
- If adapter.listTables() fails, return empty suggestions
- Wrap in try/catch, log warning but don't throw

Example error message: "ERROR: relation 'usrs' does not exist"
- Extract: 'usrs'
- Call listTables(): ['users', 'users_old', 'user_roles', ...]
- Distances: users=1, users_old=3, user_roles=6
- Return: { suggestions: ['users'], tables: ['users', 'users_old', ...] }
